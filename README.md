# Jap2Hieroglyph

## 目的と詳細

- 日本語とヒエログリフの相互変換を可能にする。

- 本来ヒエログリフと日本語は対応関係にないが、ローマ字と日本語の一対一関係を利用して相互変換を実現した。

## 仕様

- ヒエログリフに対応していない文字はそのまま出力されます。

- "ゆ"に対応するヒエログリフ"𓇌𓅱"は"いう"に変換される。  
    コードポイントが被ることによる問題である。

    ```cs
    // in Japanese and Hieroglyph correspondence dictionary
    { "い", "\U000131cc" },
    { "う", "\U00013171" },
                .
                .
                .
    { "ゆ", "\U00131cc\U0013171" }
    ```
